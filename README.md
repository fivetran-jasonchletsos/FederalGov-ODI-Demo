# FederalGov-ODI-Demo — Bureau of Federal Outcomes (BFO)

Open Data Infrastructure (ODI) reference demo for a fictional federal civilian agency. Built on Fivetran connectors, Snowflake GovCloud, Apache Iceberg, dbt, and React.

- Live: https://fivetran-jasonchletsos.github.io/FederalGov-ODI-Demo/
- App code: `bfo-app/frontend/`
- Deploy workflow: `.github/workflows/deploy.yml`

Synthetic data only. BFO is a fictional agency. No real federal data is depicted.
